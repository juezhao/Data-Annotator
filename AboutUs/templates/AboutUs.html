Goal:
The project our group will be working on for R&D mainly evolves around data annotation and its functionalities.
Data annotation can be described as the process which involves labeling the available data in various different formats
such as videos, texts and images. Data annotation, annotates or labels the content for the object of interest while
ensuring the accuracy to make sure it can be recognized by machines through computer vision. The goal for our team was creating
an application that allows the users to perform basic data annotation tasks.
